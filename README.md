A chatbot style resume page developed using [react-simple-chatbot](https://github.com/LucasBassetti/react-simple-chatbot) and [nextjs](https://github.com/zeit/next.js/)

## Instruction

**install**

```
npm install
```

**development**

```
npm run dev
```

go to browser http://localhost:3000

**production**

```
export PORT=8000
npm run build
npm run start
```
